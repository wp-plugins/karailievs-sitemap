== Description ==

This plugin adds a sitemap to your blog.
It's used to show all your pages and posts to the search engines like Google.

== Installation ==

1. Upload `karailievs-sitemap` folder to the `/wp-content/plugins/` directory
1. Move `/wp-content/plugins/karailievs-sitemap/sitemap.xml.php` file to the plugin's root directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Turn the plugin on via the Settings -> Sitemap screen
1. Open your sitemap to test it (e.g. http://myblog.mysite.com/sitemap.xml.php)
1. In your Google Webmaster Tools account add the sitemap
